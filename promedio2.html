SEMANA 2

En esta sesion aprendimos a realizar alguna etiquetas en HTML

Introduccion a HTML

<!DOCTYPE html>: Define el inicio y el final del documento HTML.

<html>: Define el inicio y el final del documento HTML.

<head>: Contiene información sobre el documento, como el título de la página y enlaces a archivos CSS o JavaScript.

<title>: Define el título de la página que se muestra en la barra de título del navegador.

Mas etiquetas HTML

<script>: Permite la inclusión de scripts JavaScript en la página.

<body>: Contiene el contenido visible de la página web, como texto, imágenes, enlaces y otros elementos.

<h1>, <h2>, <h3>, ... <h6>: Definen encabezados de diferentes niveles, donde <h1> es el más importante y <h6> el menos importante.

<p>: Crea párrafos de texto.

<a>: Crea enlaces a otras páginas web o recursos.
