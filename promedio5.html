SEMANA 5

En esta sesion aprendimos mas propiedades de CSS aplicadas en HTML

Algunas Propiedades mas de CSS

text-align: Define la alineación del texto, como "left" (izquierda), "center" (centro) o "right" (derecha).

background-color: Establece el color de fondo de un elemento.

margin: Controla el espacio exterior alrededor de un elemento.

padding: Define el espacio interior dentro de un elemento.

border: Crea un borde alrededor de un elemento.

Por aqui las ultimas Propiedades CSS

display: Controla cómo se muestra un elemento, como "block" (bloque) o "inline" (en línea).

position: Determina la posición de un elemento, como "relative" (relativa) o "absolute" (absoluta).

float: Controla el flujo de elementos a la izquierda o la derecha de un contenedor.

opacity: Define la opacidad de un elemento, donde 1 es completamente visible y 0 es completamente transparente.
