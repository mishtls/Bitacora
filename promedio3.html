MAS ETIQUETAS DE HTML

las etiquetas HTML determinan cómo se verá un sitio web. Cada una indica y representa un atributo que los navegadores deben interpretar y mostrar al usuario cuando visita cualquier sitio en internet

Algunas otras Etiquetas HTML

<img>: Inserta imágenes en la página.

<ul> y <ol>: Crean listas sin ordenar (viñetas) y listas ordenadas

<li>: Define elementos de lista dentro de <ul> o <ol>.

<div>: Se utiliza como contenedor genérico para agrupar y dar formato.

<span>: Se utiliza para aplicar estilos o scripts.

<br>: Crea un salto de línea dentro del texto.
